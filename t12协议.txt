t12API协议 v0.13

基础约定
1./static为静态目录
图片文件放入/static/img
js文件放入/static/js
css文件放入/static/css

扩展基础约定
用户头像放入img/UserPic/headPic 命名方式为UserName.xxx

协议约定
一.UserController

1.
来源/api/users/signup [get]
request: null
response：
显示注册页面signup.html

2.
来源signup.html
request:from表单
action /api/users/signup
method：post
input name：
username 用户名
password 用户密码 
truename 真实姓名
stunum 学号
school 学校 
schoolex 额外学校信息，比如校区或者学院
email 邮箱
response：
 {
	"err"  //返回错误信息   
	"data"  //如果没有错误返回额外信息
}

3.
来源/api/users/login [get]
request: null
response：
显示登陆页面login.html

4.
来源login.html
request:from表单
action /api/users/login
method：post
input name：
username 用户名
password 用户密码 
response：
 {
	"err"  :"登陆失败，用户名或密码错误" 
	"data" :"登陆成功"
}

5.
来源/api/froms/changs/add [get]
request: null
response：
显示添加项目页面 名字我还没想好你看着合适就取个吧.html

6.
来源 名字我还没想好你看着合适就取个吧.html
request:ajax 
method：post
url:/api/froms/changs/add
 {
	title      项目标题
	content    项目详细内容
	tags    [] 标签
	beginDay     开始日期   
	endDay       结束日期 
	status       状态      //状态有正常，结束，过期，关闭,待审核
	totalNeedPeopleNum 总共需要的人数
	PeoPleList  [
	{
		peopleType       需要的人类型       
		needNumber       需要的人数         
	}
	]
}
7.
来源任何登陆后的界面
request:ajax 
method：post
url:/api/users/nowusercore
response：
{
	“userName” 用户名
	"userPic"   用户头像名称
}
协议未稳定随时可能更改
